# Shortcuts

## Navigation Shortcuts

- `cmd+0` Sidebar
- `cmd+1` Code
- `cmd+2` Code Column 2
- `ctrl+1` Tab

- `cmd+shift+e` Explorer (File)
- `ctrl+shift+g` Git

## In Sidebar

- Vim navigation
- `l` (right) to open a file
- `cmd+down` to open a file (apparently works in finde)

## In Code

- Vim navigation
- `cmt+opt+.` close a tag

## Settings and Configuration

- `cmd+,` Open settings
